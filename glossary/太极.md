# 太极拳词表

## 一句话上下文

传统太极拳教学/演讲文稿，可能涉及陈、杨、吴、武、孙五大流派，常出现拳式名称（招式）、劲法、心法、步法、桩法等术语。

## 专有名词（保持原样）

### 流派
- 陈式太极拳 / 陈氏太极拳
- 杨式太极拳 / 杨氏太极拳
- 吴式太极拳 / 吴氏太极拳
- 武式太极拳 / 武氏太极拳
- 孙式太极拳 / 孙氏太极拳

### 心法 / 要领
- 太极学 / 无级学（无极学）
- 八门五步、含胸塌腰、虚领顶劲、气沉丹田、沉肩坠肘
- 沾粘连随、舍己从人、懂劲、运化转关、阴阳混一、双重
- 顶匾丢抗、缠丝劲、腰裆

### 八法（劲）

- 掤劲、捋劲、挤劲、按劲、采劲、列劲、肘劲、靠劲

### 桩 / 起势
